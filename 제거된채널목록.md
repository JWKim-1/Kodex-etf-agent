# 구조적으로 불가한 채널 제거 목록

아래 채널들은 기술적으로 개선 불가능한 이유로 Agent에서 제거됨.
API 확보나 로그인 정보가 생기면 재검토 가능.

---

## 제거된 채널

### Instagram (삼성증권)
- **제거 이유**: 봇 탐지 + 공식 API 없음. 로그인 없이는 완전 불가.
- **재검토 조건**: Instagram Graph API 공식 계정 권한 취득 시

### ~~카카오톡 채널 (삼성자산운용)~~ → ✅ 복구됨
- **해결**: rocket-web 내부 API () 직접 호출로 비로그인 수집 가능 확인

### 구글 트렌드
- **제거 이유**: 공식 API 없음. 비공식 pytrends 라이브러리 사용했으나 구글이 봇 차단.
- **재검토 조건**: 구글이 공식 Trends API 출시 시

### 삼성증권 홈페이지 이벤트 탭
- **제거 이유**: SPA 구조 + 봇 탐지. Selenium으로도 콘텐츠 추출 불가.
- **참고**: 삼성자산운용 이벤트 페이지(samsungfund.com)로 대체 운영 중
- **재검토 조건**: 로그인 기반 API 또는 홈페이지 구조 변경 시

### 퇴직연금 상품가이드 PDF
- **제거 이유**: SPA + 로그인 필요. 비로그인 접근 불가.
- **재검토 조건**: 공개 PDF URL 확보 시
