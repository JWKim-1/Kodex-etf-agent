## 방법론 핵심

| 항목 | 증권 | 개인(매스) | 은행 |
|---|---|---|---|
| 기준 컬럼 | 금융투자 (LP 감지 시 개인으로 전환) | 개인 | 은행 |
| **베이스라인** | **8주** | **8주** | **8주** |
| 계산 모듈 | `did_calculator.py` 공용 | `did_calculator.py` 공용 | `agents/bank/analyzer.py` 자체 |
| LP 노이즈 감지 | O | X | O |
| 1단계 지표 | DiD (정규화 절대변화) | DiD | DiD |
| 2단계 지표 | Z-score (ZSCORE_WINDOW=15) | Z-score (ZSCORE_WINDOW=15) | Z-score (16주 rolling) |
| 최종 점수 | sigmoid 0~100점 | sigmoid 0~100점 | sigmoid 0~100점 |
| 라플라스 α | +1,000,000 (천원단위, 1백만) | 동일 | 동일 |
| AUM 폴백 | 베이스라인 8주 미만 시 | 동일 | 동일 |
| 결과 DB | `did_history.parquet` | `did_history.parquet` | `bank_zscore_history.parquet` |

**공식**: `(현재값 - N주평균) ÷ (N주 평균절댓값 + 1,000,000)`
**라플라스는 반드시 `+`로 적용** (`max()` 금지)
**베이스라인: 증권/개인/은행 모두 8주** (과거 인수인계문서의 4주는 잘못된 정보)

### 2단계 Z-score
```
Z = (이번주 DiD - 과거 ZSCORE_WINDOW주 DiD 평균) / (과거 DiD 표준편차 + 라플라스)
score = 100 / (1 + exp(-Z * 1.5))
```
- ZSCORE_WINDOW만큼 없어도 있는 이력으로 계산 (버그 아님)
- 이력 4주 미만 → Z=None, 50점 반환
- 라플라스 보정: std<0.05면 0.05 적용 (극단값 방지)

### AUM 상대강도 폴백 (베이스라인 8주 미만 신규상장)
```
KODEX비율 = 순매수/AUM(시총)
경쟁사비율 = 순매수/AUM(시총)  
AUM DiD = KODEX비율 - 경쟁사평균비율
```
경쟁사도 없으면 절대수준 점수 + "⚠️ 경쟁사 없음 — 직접 비교 불가" 경고

## 수집기 현황

### 증권 채널 (collector.py collect_all)
삼성/미래에셋/키움/토스/한투/신한/KB 유튜브+블로그+이벤트 + 카카오채널 5개 + 구글트렌드+네이버데이터랩

### ETF 운용사 채널 (collect_all_competitor = 개인+경쟁사 공용)
KODEX/TIGER/ACE/RISE/HANARO/SOL/PLUS 유튜브+이벤트+블로그+카카오 + 뉴스

### 은행 채널 (agents/bank/collector.py)
KB/신한/하나/우리/농협 유튜브+블로그+카카오

### Selenium 수집기 (_selenium_event_collect 헬퍼)
TIGER/HANARO/ACE 이벤트 페이지 SPA 처리 (headless Chrome)
신한투자증권/KB증권 이벤트도 Selenium

### 날짜 필터
모든 이벤트 수집기: `week_start` 기준 이미 종료된 이벤트 제외
(종료일 < week_start 이면 skip)

### 수집 불가 채널 (구조적 한계)
- Instagram: Rate limit + Graph API 필요 (.instagram_session 있음)
- 구글 트렌드: Selenium도 429 → RSS+네이버데이터랩으로 대체
- 삼성증권 홈페이지 이벤트: BOT 감지 → 제거됨
- 퇴직연금 PDF: 로그인 필요 → 제거됨

## LLM 분석 구조

`scheduled_collect.extract_events()`: 4개 세션 공통
1. **키워드 1차 필터** → 관련 없는 채널 LLM 스킵
2. **모든 소스 수집** (raw_text + videos + event_details + articles 동시에)
3. **LLM 2차 판별** → 진짜 마케팅 활동인지 확인 + ETF 특정
4. 이벤트만 아니라 **모든 마케팅 활동** (유튜브 영상, 콘텐츠 등) 포함

경쟁사 LLM이 KODEX만 뽑으면 → keyword_fallback으로 자동 대체

- **`analyzer.py` 루트 수정 금지** — securities 재수출 래퍼, 실제 구현은 agents/securities/analyzer.py
